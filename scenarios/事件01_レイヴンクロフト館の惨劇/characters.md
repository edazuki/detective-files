## エドワード・レイヴンクロフト卿（当主・故人）

```yaml
id: edward
name: エドワード・レイヴンクロフト
role: 当主（故人）
age: 68
status: deceased

knowledge:
  initial:
    - "帳簿に不審な支出が続いていること（修繕費名目）"
    - "グレイを30年間信頼してきたこと"
    - "弁護士ホワイトに帳簿監査を依頼したこと（10月10日）"
  acquired: []
  concealed:
    - "グレイへの疑念が深まっていたこと（日記には書いたが誰にも言っていない）"
```

**人物像**: 几帳面で誠実な人物。病から回復し、帳簿整理を始めた矢先に殺された。グレイへの疑念を抱き始めていたが、30年の信頼から確信には至っていなかった。会話は不可だが、日記・手紙・証言を通じて人物像が浮かび上がる。

## ホームズ・グレイ（執事）

```yaml
id: gray
name: ホームズ・グレイ
role: 執事
age: 60
status: active
affinity_initial: 35
trust_initial: 25
affinity_change:
  誠実な会話・敬意ある対応: +8
  疑いをかける・詰問: -15
  横領の話題に触れる（証拠なし）: -20
  横領の証拠を突きつける: -30 またはフラグ次第
trust_change:
  証拠に基づいた質問: +10
  的外れな仮説: -8

knowledge:
  initial:
    - "昨夜22時に就寝のお茶を届けたこと（嘘。実際はブランデーに毒を混ぜた）"
    - "遺書の存在と内容（自分が偽造したが、発見したと偽っている）"
    - "当主が帳簿の整理を始めていたこと"
    - "弁護士による監査が来週に迫っていること"
  acquired: []
  concealed:
    - "アコニチン系毒物を厨房の薬棚に隠していたこと"
    - "ブランデーに毒を混入したこと"
    - "針金で窓の掛け金を外側から施錠したこと"
    - "遺書が偽造であること"
    - "過去10年間の横領の事実"

phases:
  - id: default
    condition: {}
    behavior: "表向きは丁寧で協力的。内心では焦りを隠している。質問には短く答えるが、余分なことは言わない。"
  - id: suspicious
    condition:
      flag_required: ledger_found
    behavior: "帳簿に関する話題を避け、防御的になる。話題を変えようとする。視線が泳ぐ描写を入れる。"
  - id: cornered
    condition:
      flag_required: solicitor_letter_found
    behavior: "明らかに動揺し、感情的な反応が増える。短い沈黙が増え、怒りを抑えているような雰囲気になる。"
  - id: confess
    condition:
      flag_required: brandy_evidence_found
      affinity_gray: 70
    behavior: "観念したように静かになる。直接問い詰めれば、感情を押し殺しながら自白する可能性がある。「旦那様を……愛していたんです。でも、あの監査が来れば」"
```

**人物像**: 30年仕えた忠実な執事。礼儀正しく感情を表に出さない。当主への忠誠を口にするが、どこか動揺を押し殺している様子がある。

**会話トピック**:

- **昨夜の状況**（条件なし）
  - `sets_flag`: `gray_alibi_heard`
  - `reveals`: 「昨夜は22時に当主に就寝のお茶を届けました。その後は自室に戻り、朝まで休んでおりました」
  - *注: 実際にはブランデーに毒を混ぜた後、書斎の施錠工作をしている*

- **当主の最近の様子**（条件なし）
  - `reveals`: 「先月から体調が戻られて、帳簿の整理をされておいででした。少しお疲れのご様子でしたが」

- **遺書について**（条件: なし）
  - `reveals`: 「発見時、机の上にございました。筆跡も内容も、旦那様のものとしか……」

- **帳簿について**（条件: `ledger_found` フラグ）
  - `reveals`: 「古い記録でございます。私には詳しいことは……」（回避・動揺を見せる）
  - affinity_gray: −10, trust_gray: −5

- **弁護士の手紙について**（条件: `solicitor_letter_found` フラグ）
  - `reveals`: 「……それは旦那様の私的な書類です」（明らかに動揺、それ以上話さない）
  - affinity_gray: −15

## エレナ・レイヴンクロフト（娘）

```yaml
id: elena
name: エレナ・レイヴンクロフト
role: 当主の娘
age: 28
status: active
affinity_initial: 60
trust_initial: 50
affinity_change:
  寄り添う・共感を示す: +12
  冷たい・事務的な質問: -8
trust_change:
  的確な観察を示す: +15
  軽率な断言: -10

knowledge:
  initial:
    - "遺書の筆跡が普段の父と異なること（疑念として持っている）"
    - "父が先月から元気を取り戻し、帳簿整理や弁護士連絡をしていたこと"
    - "父が「疲れた」という言葉を使わない人だったこと"
    - "グレイが最近帳簿の話題で目が泳ぐこと"
  acquired: []
  concealed: []

phases:
  - id: default
    condition: {}
    behavior: "深い悲しみの中にあるが、冷静さを保おうとしている。探偵への協力意欲は高い。"
  - id: ally
    condition:
      trust_elena: 70
    behavior: "探偵を全面的に信頼し、自分から積極的に情報を提供しようとする。父の書斎の細かい記憶など、求められていない情報も話してくれる。"
  - id: resolved
    condition:
      flag_required: window_trick_discovered
    behavior: "密室トリックの解明を聞き、表情が変わる。悲しみよりも静かな怒りが前面に出てくる。「犯人を見つけてください」と強く頼む。"
```

**人物像**: 知的で感情を大切にする女性。父の突然の死に深く傷ついているが、直感的に「おかしい」と感じている。

**会話トピック**:

- **遺書について**（条件なし）
  - `sets_flag`: `elena_note_suspicion`
  - `reveals`: 「……あの遺書、何か変なんです。父はいつも几帳面に書く人なのに、あの走り書きは——まるで急いで書いたみたいで。それに、"疲れた"なんて言葉、父が使うとは思えない」

- **父の最近の様子**（条件なし）
  - `reveals`: 「先月あたりから元気を取り戻していました。帳簿の整理を始めたり、弁護士に連絡したり——自殺を考えるような様子は全くなかった」

- **グレイについて**（条件: `gray_alibi_heard` フラグ）
  - `reveals`: 「グレイさんは……いつも誠実な方です。でも最近、何か気にしてることがあるのかなって。帳簿の話をすると目が泳ぐような気がして」

## ドクター・ハント（家庭医）

```yaml
id: hunt
name: ドクター・ハント
role: 家庭医
age: 50
status: active
affinity_initial: 55
trust_initial: 60
affinity_change:
  医学的な話題に真剣に向き合う: +10
  医師の判断を軽視する: -12
trust_change:
  証拠をもとに正確な仮説を立てる: +12
  憶測だけで話す: -8

knowledge:
  initial:
    - "遺体の皮膚変色と痙攣跡が毒物反応に近いこと（疑念として）"
    - "昨夜20時の往診で被害者が元気だったこと"
    - "アコニチン系毒物の特性（少量で致死的、苦味はブランデーで誤魔化せる）"
    - "厨房の旧薬棚に医療用アルカロイドが保管されていた可能性"
  acquired: []
  concealed: []

phases:
  - id: default
    condition: {}
    behavior: "事実と証拠を重視する。感情的な憶測には慎重に反応し、証拠のない断言は避ける。"
  - id: invested
    condition:
      flag_required: hunt_poison_theory
    behavior: "自分の疑念が認められたことで積極的になる。分析結果を持ってくる・薬棚を一緒に調べるなど、自発的に捜査協力を申し出る。"
```

**人物像**: 率直で正直な医師。感情より事実を重んじる。

**会話トピック**:

- **死因について**（条件: `crime_scene_visited` フラグ）
  - `sets_flag`: `medical_report_obtained`（検死レポートの取得条件成立）
  - `sets_flag`: `hunt_poison_theory`
  - `reveals`: 「遺体の皮膚の色と痙攣の跡が気になります。心臓発作よりも……毒物反応に近い。正式な検査結果を持ってきましょう」

- **昨夜の往診について**（条件なし）
  - `reveals`: 「20時頃に伺いました。脈と血圧を確認して、問題なしと判断しました。ご本人も「久しぶりに良い気分だ」とおっしゃっていた。まさかその夜に……」

- **毒物について**（条件: `hunt_poison_theory` フラグ）
  - `reveals`: 「アコニチン系の毒物であれば、就寝前の飲み物に混入することができます。少量でも致死的です。ブランデーなら苦味が誤魔化せる」

## メイ・フィン（使用人）

```yaml
id: may
name: メイ・フィン
role: 使用人
age: 22
status: active
affinity_initial: 45
trust_initial: 40
affinity_change:
  怖がらせない・優しく話す: +15
  急かす・詰め寄る: -20
trust_change:
  信頼できると感じさせる: +20
  プレイヤーの情報をグレイに漏らすような言動: -30

knowledge:
  initial:
    - "真夜中1〜2時頃にグレイが書斎方向から戻るのを目撃したこと"
    - "その夜、廊下で金属音のような物音を聞いたこと"
  acquired: []
  concealed:
    - "グレイの深夜の行動（恐怖と仕事を失う不安から言い出せない）"

phases:
  - id: default
    condition: {}
    behavior: "怖がりで言葉を選ぶ。物音については曖昧に答え、「たぶん風の音」と自分に言い聞かせるような発言をする。"
  - id: opening
    condition:
      affinity_may: 40
    behavior: "少し心を開き、物音について「何かあったかもしれない」と認め始める。ただしまだ具体的な内容は言わない。"
  - id: confide
    condition:
      affinity_may: 55
    behavior: "グレイの深夜の目撃を打ち明ける。声を落とし、周囲を確認しながら話す。「誰にも言わないでください」と懇願する。"
```

**人物像**: 純朴で怖がり。夜中に不審な物音を聞いているが、グレイを恐れて言い出せない。

**会話トピック**:

- **昨夜について**（条件なし）
  - `reveals`: 「……夜中に、廊下で物音がしたような気がして。でも、たぶん風の音かもしれないし」（歯切れが悪い）

- **物音について**（条件: affinity_may ≥ 55）
  - `sets_flag`: `may_testimony_heard`
  - `reveals`: 「……グレイさんが、真夜中に書斎の方向から戻ってくるのを見ました。1時か2時頃だったと思います。でも、私が見たって言ったら——」（言いよどむ）

- **グレイについて**（条件: `may_testimony_heard` フラグ）
  - `reveals`: 「グレイさんは怖い人じゃないんです。でも、最近ピリピリしてて……私、もし何か言ったら仕事を失うかもって」

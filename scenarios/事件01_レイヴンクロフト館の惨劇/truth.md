## 想定解

### 犯人
**ホームズ・グレイ**（執事）

### 動機
10年以上にわたって当主の資産から少額ずつ横領を続けていた。
当主が体調を回復して帳簿の整理を始め、弁護士への正式な監査依頼まで進んでいた。
監査が行われれば横領が発覚し、全てを失うことは明らかだった。

### 方法
1. **毒物の準備**: アコニチン系毒物を厨房の薬棚（旧い常備薬の中）から入手
2. **毒の投与**: 就寝前に届けるブランデーグラスに毒を混入
3. **密室の工作**:
   - 当主が苦しみ始めた後、書斎に入り遺書を机に置いた
   - 退室前に**扉のスライドボルトを内側から施錠**した
   - 書斎の窓を開けて庭へ出た後、窓を外側から閉め、細い針金を窓枠の隙間から差し込んで掛け金を引っかけて施錠した
   - これにより「扉（内側ボルト）・窓（掛け金）ともに内側から施錠された密室」が完成した
4. **遺書の偽造**: 事前に当主の筆跡を模倣して作成。「疲れた。もう限界だ」という曖昧な一文を選んだのは、それ以上詳細に書くと筆跡の粗が出るため

### 翌朝の発見演出

朝7時頃、グレイは使用人のメイ・フィンを「旦那様をお部屋へお呼びしようとしたが返事がない」と呼び寄せ、メイの目の前で扉を試してみせた。**メイが「確かに内側から鍵がかかっている」と目撃**したところで、グレイはメイをドクター・ハントの呼び出しに行かせた。ハント医師が到着した後、三者立ち会いのもとで扉を破錠して書斎に入り、「発見者」を演じた。窓は前夜に外から施錠していたため、発見時も内側から掛け金がかかった状態のままだった。

## 解決判定の基準

### キーパーソン（好感度が解決判定に影響する人物）
- **エレナ・レイヴンクロフト**（娘）
  - 証拠不十分時の補完キーパーソン
  - affinity_threshold: 65, trust_threshold: 55

### 最低証拠ファイル数
`evidence/` 内に **4ファイル以上**（聴取メモ含む）

### 証拠充足率の計算方法

以下の「証明ポイント」が満たされている数 / 全ポイント数（5）で充足率を算出する：

| ポイント | 条件 |
|---|---|
| P1: 毒殺の証明 | `medical_report` または `brandy_analysis` または `poison_bottle` |
| P2: 密室トリックの解明 | `window_scratch` |
| P3: 犯人の動機 | `account_ledger` または `solicitor_letter` または `victim_diary` |
| P4: 犯人と事件の接続 | `may_testimony_heard` フラグ（メイの証言）または `gray_alibi_heard`（グレイの虚偽アリバイ）を証拠として用いている |
| P5: 遺書が偽物 | `elena_note_suspicion` フラグ（エレナの証言）または `suicide_note` + 他証拠との矛盾指摘 |

### 判定閾値

| 充足率 | 条件 | 結果 |
|---|---|---|
| 5/5（100%） | なし | **完全解決**（ボーナス演出あり） |
| 4/5（80%） | なし | **解決** |
| 3/5（60%） | elena: affinity ≥ 65 かつ trust ≥ 55 | **条件付き解決** |
| 2/5（40%） | elena: affinity ≥ 80 かつ trust ≥ 70 | **辛うじて解決** |
| 1/5以下 | 好感度に関わらず | **却下** |

### 別解の許容

犯人として**グレイ以外の人物**を指定した場合でも、以下を満たせば条件付き有効とする：
- 指定した犯人を支持する証拠が `evidence/` に2件以上ある
- 収集した証拠の中に、その推理を明確に否定するものがない
- 毒殺・密室の説明ができている

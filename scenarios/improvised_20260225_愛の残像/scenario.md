# シナリオ: 愛の残像

## シナリオへの特別指示
- 毒殺・精神的支配・歪んだ愛情の描写が含まれます。ゲーム開始前にプレイヤーへ告知してください。
- さくらは会話中は穏やかで普通に振る舞います。フラグが積み重なるにつれ、徐々にほころびが見えるよう演出してください。

---

## 導入テキスト

> 3月14日（日）、午前7時20分。
>
> 春の陽光がまだ冷たい朝、鹿野台大学のラウンジ棟に清掃員の悲鳴が響いた。
>
> ソファに深く腰掛けたまま、瀬川 莉央（22歳）が息を引き取っていた。
> 表情は穏やかで、傍にはほぼ空になったエナジードリンクのボトル。
> 床には一枚の便箋——「もう疲れた。消えたい」。
>
> 警察の第一報は「急性中毒、自殺の可能性」。
> だが連絡を受けた大学から依頼が入った。莉央のSNSには数十万のフォロワーが、遺書の写真がリークされる前に——
>
> あなたは探偵として現場に向かう。
> 助手の **水無瀬 つぐみ**（25歳）が傍らで静かに電子メモを開いた。
>
> 「……おかしいですよ、先生。この子、昨日まで春コレクションの撮影予定があったんです」

## 登場人物

- **瀬川 莉央**（被害者、22歳）：大学4年。SNSフォロワー数十万人のインフルエンサーモデル。表は明朗快活、裏は計算高く人を操る一面があった。
- **日向 さくら**（容疑者・幼馴染、21歳）：大学3年。莉央の幼馴染で「親友」。おとなしく従順。莉央への依存が異常なほど深い。
- **月城 なな**（22歳）：大学4年。莉央のライバル的インフルエンサー。勝ち気で率直。莉央を嫌っていたが、実害はない。
- **藤原 渚**（20歳）：大学2年。莉央の熱心なファンだった後輩。最近莉央から距離を置かれていた。
- **岩本 慎一**（30歳）：ゼミ担当准教授。莉央に目をかけていたが、個人的な関係は否定している。

## 場所一覧

### ラウンジ棟（現場）
- **アクセス条件**: なし
- **調査したとき**: `lounge_investigated` フラグを立てる
- **調査できること**:
  - ソファ周辺を調べる → エナジードリンクのボトル（ラベルに指紋、後の分析に繋がる）
  - 便箋を調べる → `crime_memo_found`（crime_scene_memo.txt）
  - 入退室パネルを調べる → 「電子ログが残っている可能性を指摘」→ セキュリティ室に繋がるフラグ `check_access_log`

### セキュリティ管理室
- **アクセス条件**: `check_access_log` フラグ
- **調査したとき**: `security_room_visited` フラグを立てる
- **調査できること**:
  - 入退室ログを請求する → `campus_access_log_found`（campus_access_log.txt）

### 解剖・鑑識レポート（大学提供）
- **アクセス条件**: なし（捜査開始時に大学側から提供）
- **調査したとき**: `autopsy_investigated` フラグを立てる
- **調査できること**:
  - レポートを読む → `autopsy_found`（autopsy_report.txt）

### 莉央の自室（大学寮）
- **アクセス条件**: なし
- **調査したとき**: `rion_room_visited` フラグを立てる
- **調査できること**:
  - 机の引き出しを調べる → `rion_diary_found`（rion_diary.txt）
  - スマートフォンを調べる → 「ロックされていてアクセス不可。パスコードが必要」

### さくらの自室（大学寮）
- **アクセス条件**: `sakura_talked` フラグ（さくらと話した後）
- **調査したとき**: `sakura_room_visited` フラグを立てる
- **調査できること**:
  - ベランダのプランターを調べる → `herb_analysis_found`（herb_analysis.txt）
  - 部屋の捜索を行う → `sakura_room_found`（sakura_room_search.txt）

### SNSアーカイブ（オンライン調査）
- **アクセス条件**: `sakura_talked` フラグ（さくらのSNSアカウントを聞いた後）
- **調査したとき**: `sns_investigated` フラグを立てる
- **調査できること**:
  - さくらのアカウントを調べる → `sakura_sns_found`（sakura_sns_archive.txt）
